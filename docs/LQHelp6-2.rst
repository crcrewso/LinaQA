Running LinaQA under Linux
--------------------------

Open a terminal window. Change to the directory where you installed LinaQA. At
the prompt enter:

``linaqa.sh``

or

``linaqa.sh \path\to\dicom\image.dcm``

or

``linaqa.sh \path\to\dicom\imagedir``

Alternatively you can create a link to the application and place it on your desktop.
Refer to your system documentation for details.
